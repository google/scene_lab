28 /// A simple camera class that implements CameraInterface. If you just need a
29 /// very basic camera to use Scene Lab, you can use this one, which just
30 /// implements the barebones requirements, no stereoscopy or other fancy stuff.
31 ///
32 /// Scene Lab will use this camera by default unless you override by calling
33 /// SceneLab::SetCamera.
34 ///
35 /// By default, this camera uses right-handed coordinates.
36 class BasicCamera : public corgi::CameraInterface {
37  public:
38  BasicCamera();
39  virtual ~BasicCamera() {}
